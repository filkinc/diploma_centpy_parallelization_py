centpy_on_jax (черновой вариант)
 - __init__.py
 - core.py # Определение типов данных (Pars1d, Equation1d, ...) - "Контейнеры"
 - equations.py # "Библиотека" готовых уравнений (Бюргерс, Эйлер ...)
 - boundaries.py # Граничные условия
 - geometry.py # Функции создания сетки и обработки границ (+ добавление сетки)
 - schemes.py # Численные методы (SD2, SD3). Принимают (u, pars), возвращают du/dt
 - time_integration.py # Интеграторы (Runge-Kutta). Принимают функцию правой части.
 - solver.py # "Оркестратор", который собирает всё воедино и запускает цикл

В centpy (ООП): метод меняет внутреннее состояние объекта.
В jax_centpy (функциональное программирование): функция принимает старые данные и возвращает новые. Она чистая, не меняет ничего снаружи.
